# Pyside6Uic-Frontend
Pyside6-uic Front-End with GUI

Description:
This program is a front-end for Pyside2-uic.
You can convert a ui files to a py file with GUI.

Python Environment:
Python 3.7

Modules need to be installed:
PySide6

Installation1:
If you have not installed PySide6, excute [pip install pyside6] in command prompt.

Installation2:
After you placed files in your drive, run Pyside6UicFrontEnd.py(For convinience, use PyinstallerFrontEnd.py to convert the py file to a exe file).
Set the path of pyside6-uic.bat or pyside6-uic.exe(i.e. PYTHON_FOLDER/Library/bin/pyside6-uic.bat).
Choose ui file.
Push EXEC button to convert ui file to py file.

NOTICE : Do not put SPACE in the file path or file name. Otherwise you will get an error.
