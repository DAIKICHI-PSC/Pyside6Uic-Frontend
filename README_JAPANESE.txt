Pyside6UicFrontEnd
Pyside6-uicフロントエンドです。

説明：
このプログラムはPyside6-uicのフロントエンドです。
GUIでuiファイルをpyファイルに変換することができます。
  
Python環境：
Python 3.7

必要なモジュール：
PySide6
  
インストール1：
PySide6がインストールされてない場合は、コンソールで以下を実行して下さい。
pip install pyside6
  
インストール2：
ドライブにファイルを配置したら、Pyside6UicFrontEnd.pyを実行します（pyinstallerFrontEnd.pyを使用してpyファイルをexeファイルに変換すると便利です）。
pyside6-uic.bat、又はpyside6-uic.exeのパスを設定します（例：PYTHON_FOLDER/Library/bin/pyside6-uic.bat）。
uiファイルを選択してください。
EXECボタンを押して、uiファイルをpyファイルに変換します。

注意：エラーが発生するので、ファイルパスとファイル名にスペースを入れないでください。
